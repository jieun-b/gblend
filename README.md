# GBlend

GBlend는 Gaussian Splatting으로 복원한 외부 3D 씬을 Blender에 자동으로 불러와 정렬, 시각화하는 경량 애드온입니다.
Photogrammetry Importer에서 영감을 받아, 카메라 시각화 및 바닥 평면 정렬을 자동화하여 synthetic data 생성 과정을 간소화합니다.
